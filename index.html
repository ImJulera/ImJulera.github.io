Dark Knight

Dark Knight é um jogo 2D Multiplataforma desenvolvido em Unity.

Este projeto foi premiado com o 2º lugar na categoria Multimédia, do concurso de Projetos de Aptidão Profissional a nivel nacional de 2021, realizado pela PACTice. A linguagem utilizada para a realização deste projeto foi C#.

GymMethod

GymMethod é website realizado em HTML, CSS e PHP.

Este projeto foi desenvolvido para a disciplina de Plataformas de Sistemas de Informação.

O FrontEnd consiste numa loja onde clientes podem realizar encomendas. Já no BackEnd é possivel não só gerenciar a loja como também aulas de um ginásio. Dados esses que são enviados para uma aplicação Mobile onde os clientes podem inscreverem-se.
